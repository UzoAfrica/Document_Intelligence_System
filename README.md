# Multi-Modal Document Intelligence System

[![Python 3.9+]
[![FastAPI]
[![License: MIT]

A production-ready AI system that extracts, processes, and analyzes information from documents using state-of-the-art machine learning models.

## Features

-  OCR text extraction from PDFs and images
-  Text summarization using BART
- Named Entity Recognition
-  Document classification
-  Semantic search with vector embeddings
- REST API and web interface
-  Docker support for easy deployment

## Quick Start

### Prerequisites
- Python 3.9+
- Tesseract OCR
- Docker
